# AI Skills

Skills teaching AI agents how to drive the editor through the `spine-editor`
MCP server (see `packages/mcp-server`):

- `spine-rigging/` — bone hierarchies, attaching images, draw order, IK; coordinate conventions.
- `spine-animating/` — keyframe semantics (setup-pose offsets!), curves, timing patterns for idle/walk/attack.
- `spine-export/` — validation, Spine JSON 4.2 export, compatibility checklist.

Kept in-repo so skills version together with the MCP tools they describe.

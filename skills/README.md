# AI Skills

Skills teaching AI agents how to drive the editor through the MCP server (Phase 5):

- `spine-rigging/` — import images, build a bone hierarchy, attach images, verify via screenshots.
- `spine-animating/` — key posing, timing and easing principles, patterns for walk/idle/attack cycles.
- `spine-export/` — validate a project and export Spine JSON + texture atlas.

Empty until Phase 5; kept in-repo so skills version together with the MCP tools they describe.
